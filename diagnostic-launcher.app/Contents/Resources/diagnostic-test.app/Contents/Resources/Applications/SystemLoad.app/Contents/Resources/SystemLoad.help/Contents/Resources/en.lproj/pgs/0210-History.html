Version History

Newest items are listed first.

Release 2.1 (Build 130807)

  • Minor changes to ensure compliance with the latest OS X software design guidelines.
  • The reference manual uses a new design.

Release 2.0 (Build 120614)

  • The application has been updated to support all the latest technologies available in current operating system versions. The minimum required system is now Mac OS X 10.6.8.
  • The user interface has been revised.
  • The reference manual has been revised.

Release 1.3 (Build 090325)

  • The user interface has been enhanced to support Macintosh systems with more than 9 processors.
  • The application package has been updated to meet the latest software design guidelines. Support for Mac OS X Jaguar (10.2) has been removed.
  • The reference manual has been redesigned.

Release 1.2 (Build 060628)

By user request, the application now remembers the last load value and the number of processors to load the next time it is started.

Release 1.1 (Build 060223)

  • The application is released as a Universal Binary.
  • The reference manual has been revised.

Release 1.0 (Build 041024)

First official version of SystemLoad.
