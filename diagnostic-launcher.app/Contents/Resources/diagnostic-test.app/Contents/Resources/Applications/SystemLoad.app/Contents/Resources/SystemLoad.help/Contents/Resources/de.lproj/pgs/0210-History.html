Release 2.1 (Build 130807)

  • Kleinere Änderungen um Einhaltung der neuesten OS X-Softwaredesignrichtlinien sicherzustellen.
  • Das Referenzhandbuch verwendet ein neues Design.

Release 2.0 (Build 120614)

  • Das Programm wurde aktualisiert, um alle neuesten Techniken zu unterstützen, die in gegenwärtigen Systemversionen verfügbar sind. Das benötigte Mindestsystem ist nun Mac OS X 10.6.8.
  • Die Benutzerschnittstelle wurde überarbeitet.
  • Das Referenzhandbuch wurde überarbeitet.

Release 1.3 (Build 090325)

  • Die Benutzerschnittstelle wurde erweitert, um Macintosh-Systeme mit mehr als 9 Prozessoren unterstützen zu können.
  • Das Programmpaket wurde aktualisiert um die neuesten Software-Gestaltungsrichtlinien zu erfüllen. Unterstützung für Mac OS X Jaguar (10.2) wurde entfernt.
  • Das Benutzerhandbuch wurde neu gestaltet.

Release 1.2 (Build 060628)

Auf Benutzerwunsch behält das Programm nun den letzten Lastwert und die Anzahl der zu belastenden Prozessoren wenn es beim nächsten Mal gestartet wird.

Release 1.1 (Build 060223)

  • Die Anwendung wird als „Universal“-Programm veröffentlicht.
  • Das Referenzhandbuch würde überarbeitet.
